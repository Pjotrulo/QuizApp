# QuizApp

## Tech stack:

- React
- Typescript
- Sass
- MaterialUI
- Json Server
- Axios

## Libraries used: 

- [lodash](https://lodash.com/docs/4.17.15)
- [sweetalert2](https://sweetalert2.github.io/https://sweetalert2.github.io/)

## **Api used [triviaApi](https://the-trivia-api.com/)**

## Get Started:

**First console**
```
npm install
npm start
```
**Second console**
```
json-server ./src/database/db.json -c json-server.json
```
